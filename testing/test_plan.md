# Test Plan — Frontend — Grocery Store

## 1. Scope
Проверка функциональности и удобства использования веб-сайта магазина Grocery Store.

Страницы и функции для тестирования:
- Главная
- Каталог товаров
- Карточка товара
- Поиск товаров
- Фильтры и сортировка
- Корзина
- Checkout / оформление заказа
- Профиль пользователя / регистрация / логин
- Страница ошибок (404, 500)

## 2. Цели тестирования
### Функциональное:
- Проверить корректность поиска и фильтров
- Проверить работу корзины и изменения количества товаров
- Проверить оформление заказа до шага оплаты
- Проверить авторизацию и регистрацию
- Проверить отображение ошибок и уведомлений

### Пользовательское (Usability):
- Оценить удобство навигации
- Проверить, насколько понятен интерфейс и логика работы
- Найти типичные проблемы пользователей

## 3. Типы тестирования
- Unit / Integration (Jest + React Testing Library)
- E2E (Cypress)
- Accessibility (axe-core / cypress-axe)
- Performance (Lighthouse)
- Usability (тестирование с участниками)

## 4. Среда и инструменты
- Локальная dev-среда: `npm start` (Node v18+)
- Staging: при наличии staging URL
- Unit: Jest + React Testing Library
- E2E: Cypress
- Accessibility: axe-core
- Performance: Lighthouse
- Для записи сессий: Loom / OBS
- Bug tracking: GitHub Issues
