# Web Site of Grocery Store

Веб-сайт бакалейного магазина с современным пользовательским интерфейсом

## Описание проекта

**Web Site of Grocery Store** - это полностью готовая Frontend часть веб-сайта для бакалейного магазина. Проект представляет собой современное многостраничное приложение (MPA) с адаптивным дизайном, возможностью добавления товаров в корзину и прочим функционалом, которым обладает каждый современный онлайн магазин.

### Установка и запуск

1. **Клонирование репозитория**
    Необходимо открыть командную строку или программу, например Visual Studio Code и ввести следующие команды: 
    git clone https://github.com/ваш-username/web-site-grocery-store.git
    cd web-site-grocery-store

2. **Установка зависимостей**
    Необходимо ввести команду, которая установит все необходимые пакеты и библиотеки, указанные в
    package.json:
    npm install

3. **Запуск в режиме разработки**
    Для запуска проекта в режиме разработки необходимо ввести следующую команду:
    npm run dev
    После успешного запуска приложение будет доступно по адресу:
    http://localhost:5173

4. **Сборка для production**
    Для сборки готовой версии сайта необходимо ввести команду, которая создает оптимизированную версию приложения в папке dist/:
    npm run build

5. **Просмотр собранной версии**
    Для просмотра собранной версии необходимо ввести следующую команду, которая запустит локальный сервер:
    npm run preview

### Установка и запуск:

**Установка зависимостей**
npm install

**Запуск в режиме разработки с hot-reload**
npm run dev

**Сборка проекта для production**
npm run build

**Просмотр собранной версии**
npm run preview

**Запуск линтера для проверки кода**
npm run lint

### Технические требования

**Системные требования**

    Операционная система: Windows 10/11, macOS 10.15+, Linux (Ubuntu 18.04+)
    Node.js: версия 16.14.0 или выше
    npm: версия 8.0.0 или выше
    Браузер: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
    Оперативная память: минимум 4GB (рекомендуется 8GB)
    Свободное место на диске: минимум 1GB

**Стек технологий**

    Фреймворк: React 18+
    Сборщик: Vite 4+
    Язык программирования: JavaScript (ES6+)
    Стилизация: CSS3, возможно с использованием препроцессоров
    Система контроля версий: Git

**Ключевые зависимости**

        "react-dom": "^18.2.0", 
        "vite": "^4.0.0",
        "react-router-dom": "^6.0.0"
    }
